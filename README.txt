JQuery Scroll Events

About
-----
This plugin was created to create special events to the scrolling actions.
See details browsing http://code.google.com/p/jquery-scroll-events/

Installation
------------
Just include the Javascript file after you've included jQuery.

Something like this:
<script src="jquery.scroll.events.js"></script>

-----
